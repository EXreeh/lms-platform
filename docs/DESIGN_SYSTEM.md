# CognitiaX AI Design System

Brand identity derived from the official CognitiaX AI logo.

## Color palette

| Token | Hex | Usage |
|-------|-----|-------|
| Gold 400 | `#F7D060` | Highlights, accents |
| Gold 500 | `#FDB813` | Primary CTA, badges |
| Gold 600 | `#C5A028` | Gradient start |
| Green 500 | `#008037` | Success, links |
| Green 700 | `#004D3D` | Primary brand, buttons |
| Green 900 | `#002E1A` | Deep text, shadows |
| Green 950 | `#051C17` | Dark mode base |
| Teal 700 | `#004D40` | Logo transition accent |

## Typography

- **Headings:** Playfair Display (serif) — premium, academic
- **Body / UI:** Geist Sans — clean, Stripe/Linear-style legibility

## Gradients

- **Brand:** Gold → Forest green → Deep green (`--gradient-brand`)
- **Gold:** Light gold → Metallic → Bronze (`--gradient-gold`)

## Components

- **Primary button:** Brand gradient (green-gold)
- **Gold CTA:** Gold gradient with dark green text
- **Cards:** Subtle borders, gradient border variant for emphasis
- **Theme:** Light / Dark / System via `ThemeToggle`

## Assets

- Logo: `/public/brand/cognitiax-logo.png`
- Favicon: `src/app/icon.png` (from official logo)

## Dark mode

Charcoal base (`#0B0E14`) with emerald accents. Gold remains the action highlight.
